# 常用 minikube 命令
安装 minikube 之后，输入 minikube 就可以出现所有 minikube 支持的命令及介绍，列几个常用的
- 启动 cluster  
    ```shell
  minikube start
    ```
- 打开 dashboard
    ```shell
    minikube dashboard 
    ```
  
- 停止 cluster
  ```shell
  minikube stop
  ```
  
- 删除 cluster
  ```shell
  minikube delete
  ```
  
- 查看所有 profile  
  ```shell
  minikube porfile list
  ```
